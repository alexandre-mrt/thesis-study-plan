### 2. Bulletproofs++ — shorter and faster range proofs
- Source: IACR ePrint 2022/510 (https://eprint.iacr.org/2022/510) +
  follow-up analysis in "Bulletproofs++: Next Generation Confidential
  Transactions via Reciprocal Set Membership Arguments"
- Summary: Replaces the inner-product argument with a reciprocal
  set-membership argument, giving ~3x smaller range proofs than classic
  Bulletproofs and sub-linear verification with batched openings — the
  leading candidate for Ch 2.2 confidential TX upgrades.
- Relevance: core
- Target chapter: ch22
- Queued for: weekly update (2026-04-26)

### 4. Nova / SuperNova / HyperNova / ProtoStar folding landscape
- Source: Nova (ePrint 2021/370), SuperNova (2022/1758),
  HyperNova (2023/573), ProtoStar (2023/620); reference impl Sonobe
