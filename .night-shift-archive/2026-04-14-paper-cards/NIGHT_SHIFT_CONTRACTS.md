# Sprint Contracts — Thesis Study Plan Website

## Contract: Study Plan Website

### Acceptance Criteria
1. Page loads with dark academic design (#0f0f0f background)
2. Three day tabs in sticky nav bar, each with distinct accent color
3. Global progress bar showing % of checked resources
4. Each day shows 2 blocks with correct time labels
5. All 30+ resources rendered as cards with: title, author, type badge, time estimate
6. YouTube videos show thumbnail that loads iframe on click
7. Checkboxes persist across page reloads (localStorage)
8. Pomodoro timer visible and functional (25min work / 5min break)
9. Mobile responsive (cards stack on small screens)
10. All external links open in new tab

### Scoring
- Each criterion: 0 (missing), 0.5 (partial), 1.0 (complete)
- Pass threshold: ≥ 8.0 / 10.0, 0 critical failures
